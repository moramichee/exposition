# PHP Native Build

This folder contains a standalone PHP version of the public exhibition website.

It does not require Laravel, Composer, or a database to run.

## Run on XAMPP

1. Place `php-native/` inside `htdocs`.
2. Open `http://localhost/communiquer/php-native/`.

## Included

- Home, About, Artists, Localisation, Press, Contact
- Legal pages
- Registration form stored in `storage/registrations.jsonl`
- Press release PDF generated by native PHP
- Native admin panel at `admin/login.php`

## Native admin

- URL: `http://localhost/communiquer/php-native/admin/login.php`
- Email: `admin@communiquer.local`
- Password: `admin12345`
- Dynamic content saved in `storage/content.json`
- Uploaded images saved in `storage/uploads/`

## Not included

- Laravel database models and migrations
